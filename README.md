# Apollo Hospitals Valuation Model

A fully integrated institutional-style equity valuation model for Apollo Hospitals Enterprise Limited built using Microsoft Excel.
This project combines financial statement forecasting, discounted cash flow valuation, trading comparable analysis, sensitivity analysis, scenario modeling, and dashboard visualization into a single end-to-end valuation framework.

### Project Overview

This financial model was built to simulate the workflow used in:

- Investment Banking
- Equity Research
- Corporate Finance
- Private Equity
- Valuation Advisory

The model forecasts Apollo Hospitals’ future financial performance and derives an intrinsic equity valuation using multiple valuation methodologies.

## Features
### Fully Integrated 3-Statement Financial Model

The model includes:

- Income Statement
- Balance Sheet
- Cash Flow Statement

All statements are dynamically linked and flow together correctly through retained earnings, working capital adjustments, debt schedules, and cash balances.

### Discounted Cash Flow (DCF) Valuation

The DCF model includes:

- Revenue forecasting
- EBITDA margin forecasting
- Depreciation & amortization schedules
- Working capital forecasting
- Capital expenditure assumptions
- Unlevered Free Cash Flow (UFCF)
- WACC calculation
- Terminal value estimation
- Enterprise value and equity value derivation

### Comparable Company Analysis (Trading Comps)

Peer analysis was conducted using listed Indian hospital companies including:

- Max Healthcare
- Fortis Healthcare
- Narayana Health
- Aster DM Healthcare
- Global Health

Key valuation multiples used:

- EV/EBITDA
- EV/Revenue
- P/E Ratio

Market data sourced from:
Screener.in

### Scenario Analysis

The model incorporates:

- Best Case
- Base Case
- Worst Case

Scenarios dynamically impact:

- Revenue growth
- Margins
- Valuation outputs
- Implied share price
- Sensitivity Analysis

The model performs sensitivity analysis on:

- WACC
- Terminal Growth Rate

This helps evaluate valuation changes under different macroeconomic and market assumptions.

### Working Capital Schedule

Integrated working capital forecasting includes:

- Days Sales Outstanding (DSO)
- Inventory Days
- Accounts Payable Days

Net Working Capital (NWC) is dynamically linked into free cash flow calculations.

### Debt Schedule

The model includes:

- Opening debt balances
- Debt repayments
- Interest expense calculations
- Closing debt balances
- Depreciation Schedule

Separate depreciation forecasting was created using:

- Existing PP&E
- New capex additions
- Useful life assumptions
- Return on Invested Capital (ROIC)

ROIC analysis was incorporated to evaluate capital efficiency.

### Dashboard & Visualization

A dashboard sheet summarizes:

- Revenue growth
- EBITDA margins
- Valuation outputs
- Scenario outputs
- Key operational metrics

The dashboard is designed to mimic institutional equity research and investment banking presentation standards.
